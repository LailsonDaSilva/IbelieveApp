## 🗂 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/LailsonDaSilva/IbelieveApp.git

    # Entrar no diretório
    $ cd IbelieveApp

    # Instalar as dependências
    $ npm install

    # Iniciar o projeto
    $ expo start

```
